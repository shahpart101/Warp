# Warp
Headcount Scenario Planner for Warp — a lightweight tool that simulates resume ingestion, candidate scoring, and hiring decisions for early-stage teams. Includes a founder-oriented “Worth-It Score,” radar chart evaluation across 5 dimensions, and auto-generated insights based on funding, runway, and network leverage.
